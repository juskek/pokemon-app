## pokemon-app

### Getting started
1. `yarn add expo`
2. `yarn expo install`
3. Start app: `yarn expo start`

#### Installing packages
`yarn expo install package_name`

### Testing
#### Unit, Component, Integration
1. `yarn test`

#### E2E
1. `yarn expo prebuild`
2. `detox test --configuration ios.sim.debug`
